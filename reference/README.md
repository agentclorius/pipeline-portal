# Reference Files

These files are from Returna's internal product portal. Use them as a starting point for the pipeline portal layout.

- **Layout.astro** — Sidebar + top bar layout with dark/light toggle. Adapt nav items for pipeline portal.
- **global.css** — Full design system with dark/light mode variables, Tailwind setup. Copy and adapt.

The pipeline portal should feel like a sibling of this product portal — same design DNA, adapted for pipeline management.
